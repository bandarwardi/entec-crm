# EN TEC CRM - Admin Dashboard

Professional Admin Dashboard for managing electrical and technical services, built with Angular 18+ and PrimeNG.

## 🚀 Features

- **Advanced Dashboard**: Real-time sales statistics, performance metrics, and activity tracking.
- **Lead Management**: Efficient tracking of potential clients with search, filters, and reminder systems.
- **Customer CRM**: Comprehensive customer profiles with order history and detailed data management.
- **Real-time Chat**: Integrated communication system with:
  - Browser Popup Notifications.
  - HTTP Fallback for serverless environments.
  - Media attachment support.
- **Order System**: Full order lifecycle management, from intake to completion.
- **Automated Invoicing**: Professional PDF invoice generation.
- **Role-Based Access Control (RBAC)**: Secure access for Super Admins, Admins, and Agents.
- **Rich UI/UX**:
  - Modern Dark Mode support.
  - Multi-language (Arabic & English).
  - High-performance tables with lazy loading.

## 🛠 Tech Stack

- **Framework**: Angular 18+
- **UI Library**: [PrimeNG](https://primeng.org/)
- **Icons**: PrimeIcons
- **Logic**: NgRx Signals (Store)
- **Styling**: TailwindCSS & SCSS

## 📦 Installation

1. Clone the repository
```bash
git clone https://github.com/bandarwardi/entec-crm.git
```
2. Install dependencies
```bash
npm install
```
3. Run the development server
```bash
npm start
```

## 🚀 Deployment

The project is configured for Firebase Hosting.

```bash
npm run build
firebase deploy
```
